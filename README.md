# BASICS

Un repositorio educativo centrado en conceptos fundamentales de programación y estructuras de datos.

## 📋 Descripción

Este proyecto contiene implementaciones y ejemplos de algoritmos y estructuras de datos básicas. Es un recurso ideal para estudiantes que desean aprender y practicar conceptos esenciales de programación.

## 🛠️ Lenguajes y Tecnologías

El repositorio utiliza múltiples lenguajes de programación para demostrar diferentes paradigmas:

| Lenguaje | Porcentaje | Descripción |
|----------|-----------|------------|
| **C++** | 55.5% | Lenguaje compilado con enfoque en rendimiento y control de memoria |
| **Java** | 36.9% | Lenguaje orientado a objetos multiplataforma |
| **Python** | 5.9% | Lenguaje interpretado de alto nivel para scripts y ejemplos |
| **Mermaid** | 1.6% | Diagramas y visualizaciones |
| **C** | 0.1% | Lenguaje de programación de bajo nivel |

## 📁 Estructura del Repositorio

```
BASICS/
├── cpp/          # Implementaciones en C++
├── java/         # Implementaciones en Java
├── python/       # Scripts y ejemplos en Python
├── diagrams/     # Diagramas en Mermaid
└── README.md     # Este archivo
```

## 🎯 Temas Cubiertos

Este repositorio incluye implementaciones de:

- **Estructuras de Datos**: Arrays, Listas, Pilas, Colas, Árboles, Grafos
- **Algoritmos de Ordenamiento**: Bubble Sort, Quick Sort, Merge Sort, etc.
- **Algoritmos de Búsqueda**: Búsqueda Lineal, Búsqueda Binaria
- **Conceptos Fundamentales**: Recursión, Complejidad de Tiempo y Espacio
- **Programación Orientada a Objetos**: Clases, Herencia, Polimorfismo

## 🚀 Primeros Pasos

### Requisitos Previos

- **Para C++**: Compilador GCC o Clang, CMake (opcional)
- **Para Java**: JDK 8 o superior
- **Para Python**: Python 3.6 o superior
- **Para Mermaid**: Editor compatible o visualizador online

### Compilación y Ejecución

#### C++
```bash
g++ -o programa programa.cpp
./programa
```

#### Java
```bash
javac Programa.java
java Programa
```

#### Python
```bash
python programa.py
```

## 💡 Cómo Contribuir

Las contribuciones son bienvenidas. Si deseas mejorar alguna implementación o añadir nuevos algoritmos:

1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/nuevo-algoritmo`)
3. Commit tus cambios (`git commit -m 'Añade nuevo algoritmo'`)
4. Push a la rama (`git push origin feature/nuevo-algoritmo`)
5. Abre un Pull Request
